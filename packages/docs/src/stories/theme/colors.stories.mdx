Colors

## Default Props

``` js
import { theme } from '@refract-ui/core';
const { colors } = theme({});
```

## Theme Configuration

Colors can be configured by extending the global styles via function or with static values. In each example, we replace the `colors.primary`

### Using Function Values

``` js
import { theme } from '@refract-ui/core';

const appTheme = theme({
  globalStyles: (config) => ({
    colors: {
      primary: '#333'
    }
  })
});
```

### Using Static Values

``` js
import { theme } from '@refract/core';

const appTheme = theme({
  globalStyles: {
    colors: {
      primary: '#333'
    }
  }
});
```
