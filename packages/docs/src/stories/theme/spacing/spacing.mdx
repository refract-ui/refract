## Default Props

``` js
import { theme } from '@refract-ui/core';
const { spacing } = theme({});

assert(spacing).isEqual({
  0: "0rem",
  1: "0.25rem",
  2: "0.5rem",
  3: "1rem",
  4: "1.5rem",
  5: "3rem",
  basis: 1
});
```

## Theme Configuration

`spacing` can be configured by extending the global styles via function or with static values. In each example, we replace the `spacing[3]` with a new value and extend the `spacing` object with a new key-value pair.

### Using Function Values

<Story
  name="Spacing Function"
  id="core-theme-spacing--function-template"
>
  {FunctionTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { spacing } = theme({
  spacing: ({ defaults }) => ({
    ...defaults,
    3: '1.25rem',
    6: '6rem'
  })
});
```

### Using Static Values

<Story
  name="Spacing Static"
  id="core-theme-spacing--static-template"
>
  {StaticTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { spacing } = theme({
  spacing: {
    3: '1.25rem',
    6: '6rem'
  }
});
```
