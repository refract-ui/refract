# Borders

The `borders` object includes a set of attributes for constructing borders at various breakpoints. This includes things like `borderWidth`, `broderStyle`, `borderColor`, and `borderRadius`.

## Theme Configuration

`broders` can be configured by extending the global styles via function or with static values. In each example, we replace the `borders.md.borderRadius` with a new value and extend the `borders.lg` object with a new key-value pair.

### Using Function Values

``` js
import { theme } from '@refract-ui/core';

const { borders } = theme({
  borders: ({ defaults }) => ({
    ...defaults,
    md: {
      ...defaults.md,
      borderWidth: '4px'
    },
    lg: {
      ...defaults.lg,
      borderWidth: '1px',
      borderStyle: 'solid',
    }
  })
});
```
