## Theme Configuration

`darkColors` can be configured by extending the global styles via function or with static values. In each example, we replace the `darkColors.primary` with a new color and extend the `darkColors` object with a new key-value pair.

### Using Function Values

<Canvas showCode>
  <Story name="colorsFunction">
    {FunctionTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { darkColors } = theme({
  darkColors: ({ defaults }) => ({
    ...defaults,
    white: '#fefefe',
    primary: 'purple'
  })
});
```

### Using Static Values

<Canvas showCode>
  <Story name="colorsStatic">
    {StaticTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { darkColors } = theme({
  darkColors: {
    white: '#fefefe',
    primary: 'purple'
  }
});
```

### Derived from `themeColors`

`darkColors` are derived from the `themeColors`. Passing different colors to the `themeColors` object also cascades into the `darkColors`. In this example, we've updated the `themeColors.primary` value, which gives us a different `subtleColor.primary` value.

``` js
const { darkColors } = theme({
  themeColors: {
    primary: 'purple'
  }
});
```
