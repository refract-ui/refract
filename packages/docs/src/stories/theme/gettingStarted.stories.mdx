# Getting Started

Install refract

`npm i @refract-ui/core`

`yarn add @refract-ui/core`

## Button Example

Import the Refract Button component.

```js
import React from 'react';
import { Button } from '@refract-ui/core';

function HomePage() {
  return (
    <HomePage>
      <Button>Enter</Button>
    </HomePage>
  );
}
```

