# Getting Started

Install refract

`npm i @refract-ui/core`

`yarn add @refract-ui/core`

## Button Example

Import the Refract Button component:

```js
import React from 'react';
import { Button } from '@refract-ui/core';

function HomePage() {
  return (
    <HomePage>
      <Button>Enter</Button>
    </HomePage>
  );
}
```

<Canvas>
  <Story name="basic button">
    <Button>Enter</Button>
  </Story>
</Canvas>

Update the `Button` background color by passing in the `bg` prop:

```js
<Button bg="green">Green Button</Button>
```

<Canvas>
  <Story name="green button">
    <Button bg="green">Green Button</Button>
  </Story>
</Canvas>

Breakpoint specific styles can be achieved by passing in a style object into one of the media query props, like `ltSm`:

```js
<Button ltSm={{ bg: 'orange', py: '40px' }}>Orange at Sm</Button>
```

  <Story name="breakpoint sizes">
    <Button ltSm={{ bg: 'orange', py: '40px' }}>Orange at Sm</Button>
  </Story>
</Canvas>
