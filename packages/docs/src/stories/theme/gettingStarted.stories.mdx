# Getting Started

Install refract:

`npm i @refract-ui/core`

`yarn add @refract-ui/core`

## Button Example

Import the Refract Button component:

```js
import React from 'react';
import { Button } from '@refract-ui/core';

function HomePage() {
  return (
    <HomePage>
      <Button>Enter</Button>
    </HomePage>
  );
}
```

<Canvas>
  <Story name="basic button">
    <Button>Enter</Button>
  </Story>
</Canvas>

#### Button Style Props

Update the `Button` background color by passing in the `bg` prop:

```js
<Button bg="green">Green Button</Button>
```

<Canvas>
  <Story name="green button">
    <Button bg="green">Green Button</Button>
  </Story>
</Canvas>

#### Breakpoint Specific Styles

Styling a Refract component by screen size/breakpoint is easily done by passing in a style object into one of the media query props like, in this example below, the `ltSm` prop.

This button below will have an orange background and more vertical padding on small
screens:

```js
<Button ltSm={{ bg: 'orange', py: '40px' }}>Orange at Sm</Button>
```
