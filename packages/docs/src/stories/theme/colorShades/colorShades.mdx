The `colorShades` attribute set is a collection of shades and tints of the colors defined in the `colors` attribute set. A ramping function is applied so that 9 variants of each color are generated from lightest to darkest. These shades are used throughout the theme for things like hover states and border colors.

## Theme Configuration

`colorShades` can be configured by extending the global styles via function or with static values. In each example, we replace the `colorShades.gray100` with a new color and extend the `colorShades` object with a new key-value pair.

### Using Function Values

<Story
  name="colorsFunction"
  id="core-theme-colorshades--function-template"
>
  {FunctionTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { colorShades } = theme({
  colorShades: ({ defaults }) => ({
    ...defaults,
    gray100: '#fefefe',
    primary: 'purple'
  })
});
```

### Using Static Values

<Story
  name="colorsStatic"
  id="core-theme-colorshades--static-template"
>
  {StaticTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { colorShades } = theme({
  colorShades: {
    white: '#fefefe',
    primary: 'purple'
  }
});
```

### Derived from `colors`

`colorShades` are derived from the `colors`. Passing different values to the `colors` object also cascades into the `colorShades`. In this example, we've updated the `themeColors.red` value, which gives us a different set of `colorShades.redXXX` values.

<Story
  name="colorsDerive"
  id="core-theme-colorshades--derive-template"
>
  {DeriveTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { colorShades } = theme({
  themeColors: {
    red: 'tomato'
  }
});
```
