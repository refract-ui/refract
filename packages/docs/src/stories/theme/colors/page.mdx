## Default Props

``` js
import { theme } from '@refract-ui/core';
const { colors } = theme({});

assert(colors).isEqual({
  black: '#000000',
  blue: '#3978EC',
  cyan: '#17a2b8',
  gray: '#B0B9C9',
  green: '#6AD193',
  indigo: '#6610f2',
  orange: '#fd7e14',
  pink: '#d63384',
  purple: '#6f42c1',
  red: '#DE7283',
  teal: '#20c997',
  white: '#ffffff',
  yellow: '#EFC174'
});
```

## Theme Configuration

`colors` can be configured by extending the global styles via function or with static values. In each example, we replace the `colors.primary` with a new color and extend the `colors` object with a new key-value pair.

### Using Function Values

<Story
  name="colorsFunction"
  id="core-theme-colors--function-template"
>
  {FunctionTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { colors } = theme({
  colors: ({ defaults }) => ({
    ...defaults,
    white: '#fefefe',
    brand: 'purple'
  })
});
```

### Using Static Values

<Story
  name="colorsStatic"
  id="core-theme-colors--static-template"
>
  {StaticTemplate.bind({})}
</Story>

``` js
import { theme } from '@refract-ui/core';

const { colors } = theme({
  colors: {
    white: '#fefefe',
    brand: 'purple'
  }
});
```
