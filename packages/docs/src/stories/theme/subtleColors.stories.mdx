## Theme Configuration

`subtleColors` can be configured by extending the global styles via function or with static values. In each example, we replace the `subtleColors.primary` with a new color and extend the `subtleColors` object with a new key-value pair.

### Using Function Values

<Canvas showCode>
  <Story name="colorsFunction">
    {FunctionTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { subtleColors } = theme({
  subtleColors: ({ defaults }) => ({
    ...defaults,
    white: '#fefefe',
    primary: 'purple'
  })
});
```

### Using Static Values

<Canvas showCode>
  <Story name="colorsStatic">
    {StaticTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { subtleColors } = theme({
  subtleColors: {
    white: '#fefefe',
    primary: 'purple'
  }
});
```

### Derived from `themeColors`

`subtleColors` are derived from the `themeColors`. Passing different colors to the `themeColors` object also cascades into the `subtleColors`. In this example, we've updated the `themeColors.primary` value, which gives us a different `subtleColors.primary` value.

``` js
const { subtleColors } = theme({
  themeColors: {
    primary: 'purple'
  }
});
```
