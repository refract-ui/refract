# Theme Color Shades

The `themeColorShades` attribute set is a collection of shades and tints of the colors defined in the `themeColors` attribute set. A ramping function is applied so that 9 variants of each color are generated from lightest to darkest. These shades are used throughout the theme for things like hover states and border colors of elements that use the `themeColors` values.

## Default Props

``` js
import { theme } from '@refract-ui/core';
const { themeColors } = theme({});

assert(themeColors).isEqual({
  danger: '#DE7283',
  dark: '#575C64',
  info: '#17a2b8',
  light: 'rgb(255, 255, 255)',
  primary: '#3978EC',
  secondary: '#B0B9C9',
  success: '#6AD193',
  warning: '#EFC174'
});
```

## Theme Configuration

`themeColorShades` can be configured by extending the global styles via function or with static values. In each example, we replace the `themeColorShades.primary` with a new color and extend the `themeColorShades` object with a new key-value pair.

### Using Function Values

<Canvas showCode>
  <Story name="colorsFunction">
    {FunctionTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { themeColors } = theme({
  themeColorShades: ({ defaults }) => ({
    ...defaults,
    white: '#fefefe',
    primary100: 'purple'
  })
});
```

### Using Static Values

<Canvas showCode>
  <Story name="colorsStatic">
    {StaticTemplate.bind({})}
  </Story>
</Canvas>

``` js
import { theme } from '@refract-ui/core';

const { themeColors } = theme({
  themeColors: {
    white: '#fefefe',
    primary100: 'purple'
  }
});
```

### Derived from `themeColors`

`themeColorShades` are derived from the `themeColors`. Passing different colors to the `themeColors` object also cascades into the `themeColorShades`. In this example, we've updated the `themeColors.primary` value, which gives us a different set of `themeColorShades.primaryXXX` values.
