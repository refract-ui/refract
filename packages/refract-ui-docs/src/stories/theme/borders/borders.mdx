## Default Props

```js
import { theme } from '@refract-ui/react';
const { borders } = theme({});

assert(borders).isEqual({
  lg: {
    borderRadius: '0.3rem'
  },
  md: {
    borderRadius: '0.5rem',
    borderStyle: 'solid',
    borderWidth: '2px'
  },
  xs: {
    borderColor: 'rgb(249, 249, 251)',
    borderRadius: '0.2rem',
    borderStyle: 'solid',
    borderWidth: '1px'
  }
});
```

## Theme Configuration

`borders` can be configured by extending the global styles via function or with static values. In each example, we replace the `borders.md.borderRadius` with a new value and extend the `borders.lg` object with a new key-value pair.

### Using Function Values

<Story name="Borders Function" id="core-theme-borders--function-template">
  {FunctionTemplate.bind({})}
</Story>

```js
import { theme } from '@refract-ui/react';

const { borders } = theme({
  borders: ({ defaults }) => ({
    ...defaults,
    md: {
      ...defaults.md,
      borderWidth: '4px'
    },
    lg: {
      ...defaults.lg,
      borderWidth: '1px',
      borderStyle: 'solid'
    }
  })
});
```

### Using Static Values

<Story name="Borders Static" id="core-theme-borders--static-template">
  {StaticTemplate.bind({})}

```js
import { theme } from '@refract-ui/react';

const { borders } = theme({
  borders: {
    md: {
      borderWidth: '2px',
      borderStyle: 'solid',
      borderRadius: '8px'
    }
  }
});
```
