The `themeColorOpacities` attribute set is a collection of opacities of the colors defined in the `themeColors` attribute set. A ramping function is applied so that 9 variants of each color are generated from lightest to darkest. These opacities are used throughout the theme for things like hover states and border colors of elements that use the `themeColors` values.

## Theme Configuration

`themeColorOpacities` can be configured by extending the global styles via function or with static values. In each example, we replace the `themeColorOpacities.primary` with a new color and extend the `themeColorOpacities` object with a new key-value pair.

### Using Function Values

<Story
  name="colorsFunction"
  id="core-theme-themecoloropacities--function-template"
>
  {FunctionTemplate.bind({})}
</Story>

```js
import { theme } from '@refract-ui/react';

const { themeColors } = theme({
  themeColorOpacities: ({ defaults }) => ({
    ...defaults,
    white: '#fefefe',
    primary100: 'purple'
  })
});
```

### Using Static Values

<Story name="colorsStatic" id="core-theme-themecoloropacities--static-template">
  {StaticTemplate.bind({})}
</Story>

```js
import { theme } from '@refract-ui/react';

const { themeColors } = theme({
  themeColors: {
    white: '#fefefe',
    primary100: 'purple'
  }
});
```

### Derived from `themeColors`

`themeColorOpacities` are derived from the `themeColors`. Passing different colors to the `themeColors` object also cascades into the `themeColorOpacities`. In this example, we've updated the `themeColors.primary` value, which gives us a different set of `themeColorOpacities.primaryXX` values.

<Story name="colorsDerive" id="core-theme-themecoloropacities--derive-template">
  {DeriveTemplate.bind({})}
</Story>

```js
import { theme } from '@refract-ui/react';

const { themeColorOpacities } = theme({
  themeColors: {
    primary: 'purple'
  }
});
```
